# Changelog

## [0.10.2] - 2026-03-07

### Исправлено

- IndentationError в client.py.
- SyntaxError в filter_bar.py: три dataclass-а Reset(Message) в TaskFilterBar, QueueFilterBar и GradebookFilterBar остались с пустыми телами, что полностью ломало импорт TUI.
- Выбор курса: OptionList.highlighted оставался None при динамическом добавлении опций через add_option(), из-за чего Enter не работал.
- Затенение MRO: type-стабы (def _show_status(...): ...) в TasksMixin, QueueMixin и GradebookMixin перекрывали реальные реализации из CoreMixin, ломая статусные сообщения, отслеживание курсора и навигацию на экран submission.
- @on обработчики: метакласс Textual обнаруживает @on декораторы только на классах, которые он создаёт напрямую. Mixin-классы были невидимы. Добавлен _register_mixin_handlers() для регистрации всех обработчиков из mixin-ов.
