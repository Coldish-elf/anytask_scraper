# Changelog

## [0.8.0] - 2026-02-27

### Добавлено

- Поддержка мультивыбора фильтров при экспорте в TUI.
- Подробная документация

### Изменено

- Внешний вид кнопок экспорта в TUI.

## [0.7.5] - 2026-02-27

### Добавлено

- Строковая фильтрация по диапазону для экспорта.
- Открытие собственного submission в курсе на странице Tasks в TUI.

### Изменено

- Обработка фокуса и навигации для новых полей ввода на вкладке экспорта.
- Рефактор размеров и пропорций панели экспорта/превью, а также отдельных элементов темы TUI.

### Исправлено

- Цвета disabled-состояний в экспорте в TUI: убраны некорректные оттенки и лишняя подсветка.

## [0.7.4] - 2026-02-25

### Добавлено

- Метод discover для получения списка курсов, с возможностью фильтрации по роли и автоматического добавления в TUI.

### Изменено

- Поведение блоков комментариев для автоматического увеличения высоты.
- Цвета для сообщений-статусов в TUI в submissions.

### Исправлено

- Ошибка, при которой gradebook в TUI не листался вниз при навигации стрелками.
- Ошибка, при которой в submissions не отображались сообщения-статусы, оставляя пустые блоки.
- Ошибка, при которой в submissions не срабатывали переносы строк в сообщениях.

## [0.7.3] - 2026-02-24

### Добавлено

- Параметр filename при экспорте.
- ActionMenu при нажатии пкм в TUI.
- Копирование содержимого TUI через шорткат и пкм.
- Параметр скачивания/пропуска файлов решений в submissions.

### Исправлено

- Ошибка, при которой параметры фильтрации не применялись.

## [0.7.2] - 2026-02-18

### Добавлено

- Первая версия документации.

### Изменено

- Обновлен README.md.
- Обновлен roadmap.md.

## [0.7.1] - 2026-02-15

### Добавлено

- Логирование.
- Поддержка debug-режима в CLI (`--debug` / `-d`) и запись логов в файл (`--log-file`).
- Параметр `debug` в пользовательских настройках.
- Поддержка запуска TUI в debug-режиме.

### Изменено

- Логирование добавлено в ключевые модули CLI, TUI, client/parser/storage/display.
- Улучшена обработка ошибок предзагрузки Export в TUI: ошибки логируются подробнее и показываются в более чистом виде.

### Исправлено

- Исправлена ошибка скачивания прикрепленных файлов, когда создавалась директория, но сам файл не скачивался.

## [0.7.0] - 2026-02-13

### Добавлено

- `settings init` теперь автоматически создает `credentials.json` с шаблоном, если файл отсутствует.
- В настройки по умолчанию при `settings init` добавлено `auto_login_session: true`.

### Изменено

- Переработана логика экспортных фильтров и предзагрузки данных для `Tasks`, `Queue`, `Submissions`, `Gradebook`.
- Добавлены динамические подписи и параметры фильтрации на странице Export в TUI.

### Исправлено

- Навигация по полям фильтров на странице Export: неактивные (`disabled`) поля пропускаются при переходе стрелками и при циклическом фокусе.
- Стабилизировано обновление фильтров и предпросмотра после загрузки данных.
- Добавлена защита от race-condition при асинхронной предзагрузке.
- Улучшено визуальное выделение фокуса в группах `Export Type` и `Format`: текущий элемент под курсором выделяется цветом без изменения размеров.
