# TUI

TUI запускается командами:

```bash
anytask-tui
# или
anytask-scraper tui
```

## Экран входа

Поддерживаются два варианта:

1. Вход по `username/password`.
2. Вход по сохранённой сессии (`.anytask_session.json`), если файл существует.

Если в настройках `auto_login_session=true` и файл сессии найден, приложение входит автоматически.

## Структура интерфейса

Главный экран разделён на две панели:

1. Слева список курсов и поле добавления Course ID.
2. Справа вкладки: `Tasks`, `Queue`, `Gradebook`, `Export`.

### Вкладка `Tasks`

Показывает задачи выбранного курса.

Режим автоматически определяется по данным курса:

- student-view (балл, статус, дедлайн)
- teacher-view (секция, максимум баллов, дедлайн).

Доступны:

- фильтры по тексту, статусу, секции
- детальная панель с описанием выбранной задачи
- открытие собственного submission по выбранной задаче (если доступно в данных issue)
- копирование выбранной записи в буфер (`Ctrl+Y` или контекстное меню).

### Вкладка `Queue`

Доступна для курсов с правами преподавателя.

Функции:

- таблица очереди (`student`, `task`, `status`, `reviewer`, `updated`, `grade`)
- фильтры по тексту, студенту, задаче, статусу, проверяющему
- сортировка и навигация по таблице
- детальная панель комментариев выбранного issue
- полноэкранный экран submission с копированием (`Ctrl+Y`) и прокруткой `j/k`.

### Вкладка `Gradebook`

Показывает таблицу по группам:

- динамические колонки задач
- суммарный балл
- цветовая индикация статусов оценок.

Есть фильтры по студенту, группе и преподавателю.

### Вкладка `Export`

Позволяет собрать экспорт без CLI-команды.

Выбираются:

- тип данных: `Tasks`, `Queue`, `Submissions`, `Gradebook`
- формат: `JSON`, `Markdown`, `CSV`, `Files Only`
- колонки через `ParameterSelector`
- фильтры строк (включая диапазон фамилий `From/To`)
- необходимость скачивания файлов submissions
- выходная папка и пользовательское имя файла.

Есть динамичный preview.

## Горячие клавиши

Глобальные:

| Клавиша | Действие |
| --- | --- |
| `Ctrl+Q` | Выход из приложения. |
| `Ctrl+C` дважды | Быстрый выход. |
| `Ctrl+L` | Logout. |
| `?` | Показать/скрыть встроенную справку. |
| `Esc` | Закрыть оверлей/вернуться назад. |
| `Ctrl+Y` | Копировать текущий выбор в буфер. |

Навигация:

| Клавиша | Действие |
| --- | --- |
| `Tab` / `Shift+Tab` | Циклический переход фокуса. |
| `1` `2` `3` `4` | Переключение вкладок (`Tasks`, `Queue`, `Gradebook`, `Export`). |
| `Up` / `Down` | Навигация стрелками вверх/вниз по таблицам, спискам и пунктам меню. |
| `Left` / `Right` | Переключение стрелками элементов управления там, где это поддерживает виджет. |
| `h` / `l` | Фокус в левую/правую панель. |
| `j` / `k` | Навигация вниз/вверх по таблицам и спискам. |

Фильтры:

| Клавиша | Действие |
| --- | --- |
| `/` | Фокус на фильтры активной вкладки. |
| `Ctrl+Up` | Перейти к фильтрам. |
| `Ctrl+Down` | Вернуться к таблице/контенту. |
| `Ctrl+Left` / `Ctrl+Right` | Переключение полей фильтра. |
| `r` | Сброс фильтров. |
| `u` | Отмена последнего сброса фильтров. |

Курсы:

| Клавиша | Действие |
| --- | --- |
| `a` | Добавить курс по ID. |
| `d` | Автообнаружение курсов из профиля. |
| `x` | Удалить выбранный курс из списка и кешей. |

## Буфер обмена

Копирование работает по платформенным backend'ам:

- macOS: `pbcopy`
- Windows: `powershell` / `pwsh` / `clip`
- Linux: `wl-copy` / `xclip` / `xsel` / `termux-clipboard-set`
- fallback через OSC52 (если поддерживается терминалом).

## Где TUI хранит данные

- Настройки: `.anytask_scraper_settings.json` в текущей директории
- Сессионный файл по умолчанию: `.anytask_session.json`
